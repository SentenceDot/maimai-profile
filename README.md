# maimai profile
利用 puppeteer 產生 maimai-net「プレイヤーデータ」截圖

* 本專案依賴 puppeteer  
使用本專案之前  
`npm install puppeteer --save` 或是  
下載 repo裡面的 `package.json` 後 `npm install`  

# 使用方法
更改 `index.js` 裡面的帳號密碼後 `node index.js`

輸出檔名預設為 `example.png`，為 `png` 圖檔
